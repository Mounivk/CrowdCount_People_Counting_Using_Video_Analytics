# CrowdCount – People Counting Using Video Analytics 

CrowdCount is a Python-based people counting system that uses **video analytics and computer vision** techniques to detect and count people in video streams. This project is useful for crowd monitoring, public safety, smart surveillance, and analytics applications.

---

## Features
- People detection using **YOLO / OpenCV**
- Counts people from **uploaded videos or live streams**
- Real-time frame processing
- Bounding box visualization
- Accurate counting using tracking logic
- Modular and easy-to-understand Python code

---

## Technologies Used
- Python 
- OpenCV
- YOLO (v8 / v5 – configurable)
- NumPy
- Pandas
- CVZone (optional)
- Torch / Ultralytics
